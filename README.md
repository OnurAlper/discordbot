# Discord Bot

## İndirilmesi Gereken Kütüphaneler
- python-dotenv
- discord.py

## Kullanışı
.env dosyası oluşturup tokeninizi ve kanal idlerini girmeniz yeterli olacaktır.


## Açıklama
Discord token ile bağlantı sağlanır.
Botu kanalınıza ekledikten sonra her günün belirli saat aralıklarında Bot tarafında mesaj gönderilmektedir.
